# MajiVoice - Quick Start Guide

## Get Your Super Admin Access in 3 Steps

### Step 1: Start the App
```bash
npm run dev
```

### Step 2: Access Setup Page
Open your browser and go to:
```
http://localhost:5173/#setup-admin
```

### Step 3: Create Admin Account
Fill in the form:
- **Email**: `admin@majivoice.ke`
- **Password**: Your secure password
- **Full Name**: Your name

Click **Create Super Admin** and you're done!
